Вывод формулы Резерфорда

При движении в центральном поле закон сохранения энергии записывается в виде:

\begin{equation} \label{eq:E} \begin{aligned} E = \frac{\mu}{2} \left( \dot r^2 + r^2 \dot{\phi}^2 \right) + U(r) = \frac{\mu \dot{r}^2}{2} + \frac{M^2}{2 \mu r^2} + U(r), \end{aligned} \end{equation}

где $M=\mu r^2\dot{\phi}$ -- момент импульса приведённой частицы. Т. к. используется приведённая частица, рассмотрение ведётся в системе центра масс. Отсюда следует, что

\begin{equation} \label{eq:drdt} \begin{aligned} \frac{dr}{dt} = \sqrt{ \frac{2}{\mu} \cdot \left( E-U(r) \right) - \frac{M^2}{\mu^2 r ^2} }. \end{aligned} \end{equation}

Т. к. $d\phi=\frac{M}{\mu r^2}dt$, то

\begin{equation} \label{eq:dphi} \begin{aligned} \phi = \int \frac{ \frac{M}{\mu r^2} dr }{ \sqrt{ \frac{2}{\mu} \left( E-U(r) \right) - \frac{M^2}{\mu^2 r ^2} } } + const. \end{aligned} \end{equation}

Определим угол между какой-либо из асимптот и биссектрисой гиперболы, являющейся траекторией приведённой частицы, как

\begin{equation} \label{eq:phi0} \begin{aligned} \phi_0 = \int_{r_{min}}^{\infty} \frac{ \frac{M}{r^2} dr }{ \sqrt{ 2 \mu \left( E-U(r) \right) - \frac{M^2}{r ^2} } }. \end{aligned} \end{equation}

В случае кулоновского потенциала $U(r)=\frac{\alpha}{r}$

\begin{equation} \label{eq:phi_step1} \begin{aligned} \phi_0 = - \int_{r_{min}}^{\infty} \frac{ d \left( \frac{M}{r} \right) }{ \sqrt{ 2 \mu E+ \left( \frac{\mu \alpha}{M} \right)^2 - \left( \frac{M}{r} +\frac{\mu \alpha}{M} \right)^2 } } \end{aligned} \end{equation}

или

\begin{equation} \label{eq:phi_step2} \begin{aligned} \phi_0 = - \int_{r_{min}}^{\infty} \frac{ d \left( \frac{ \frac{M}{r} + \frac{\mu \alpha}{M} }{ \sqrt{ 2\mu E+ \left( \frac{\mu \alpha}{M} \right)^2 } } \right) }{ \sqrt{ 1 - \frac{ \left( \frac{M}{r} +\frac{\mu \alpha}{M} \right)^2 }{ 2 \mu E + \left( \frac{\mu \alpha}{M} \right)^2 } } }. \end{aligned} \end{equation}

То есть

\begin{equation} \label{eq:phi_step3} \begin{aligned} \phi_0 = \arccos{ \left( \frac{ \frac{M}{r} + \frac{\mu \alpha}{M} }{ \sqrt{ 2 \mu E+ \left( \frac{\mu \alpha}{M} \right)^2 } } \right) \bigg|_{r_{min}}^{\infty} }. \end{aligned} \end{equation}

Т. к. в точке максимального сближения $r_{min}$ выполняется $\frac{dr}{dt}=0$, а значит, $2E-2\frac{\alpha}{r_{min}}-\frac{M^2}{\mu r_{min}^2}=0$, то $\frac{ \frac{M}{r_{min}} + \frac{\mu \alpha}{M} }{ \sqrt{ 2 \mu E + \left( \frac{\mu \alpha}{M} \right)^2 } }=1$ и $\arccos{1}=0$. Тогда

\begin{equation} \begin{aligned} \label{phi_fin} \phi_0 = \arccos{ \left( \frac{ \frac{\mu \alpha}{M} }{ \sqrt{ 2 \mu E+ \left( \frac{\mu \alpha}{M} \right)^2 } } \right) }.~~~(1) \end{aligned} \end{equation}

При использовании соотношений $E=\frac{ \mu V_{\infty}^2 }{2}$ и $M=\mu \rho V_{\infty}$, где $\rho$ -- прицельный параметр, а $V_{\infty}$ -- скорость налетающей частицы в лабораторной системе на бесконечном расстоянии от мишени, выражение (1) принимает вид

\begin{equation} \begin{aligned} \label{phi_rho} \phi_0 = \arccos{ \left( \frac{ \frac{\alpha}{ \rho V_{\infty}} }{ \sqrt{ \left( \mu V_{\infty} \right)^2+ \left( \frac{\alpha}{\rho V_{\infty}} \right)^2 } } \right) }. \end{aligned} \end{equation}

Отсюда следует, что

\begin{equation} \begin{aligned} \label{rho} \rho \left( \chi \right) = \frac{\alpha}{ \mu V_{\infty}^2 } \tan{ \phi_0 } = \frac{\alpha}{ \mu V_{\infty}^2 } \cot{ \frac{\chi}{2} }, \end{aligned} \end{equation}

т. к. угол рассеяния $\chi$ связан с $\phi_0$ как $\chi=\pi - 2 \phi_0$.

Поскольку $d\sigma=2\pi \rho(\chi) |\frac{d \rho(\chi)}{d \chi}| d \chi = \frac{\rho(\chi)}{\sin{\chi}} |\frac{d \rho(\chi)}{d \chi}| d\Omega$, где $d\Omega=2 \pi \sin{\chi} \cdot d \chi$, и

\begin{equation} \begin{aligned} \label{drhodtheta} \frac{d \rho \left( \chi \right)}{d \chi} = - \frac{1}{2} \frac{\alpha}{ \mu V_{\infty}^2 } \frac{1}{\sin^2{ \frac{\chi}{2} }} , \end{aligned} \end{equation}

то

\begin{equation} \begin{aligned} \label{dSigmadOmega} \frac{d \sigma}{d \Omega} = \left( \frac{\alpha}{ 2 \mu V_{\infty}^2 } \right)^2 \frac{1}{\sin^4{ \frac{\chi}{2} }} = \left( \frac{2 \mu \alpha}{ 4 \left( \mu V_{\infty} \right)^2 } \right)^2 \frac{1}{\sin^4{ \frac{\chi}{2} } }.~~~(2) \end{aligned} \end{equation}

В системе центра масс выражение (2) запишется как

\begin{equation} \begin{aligned} \label{dSigmadOmegaCM} \frac{d \sigma}{d \Omega_{CM}} = \left( \frac{2 \mu \alpha}{ 4 p_{CM}^2 } \right)^2 \frac{1}{\sin^4{ \frac{\chi_{CM}}{2} }},~~~(3) \end{aligned} \end{equation}

где $p_{CM}$ -- импульс в системе центра масс, равный $p_{CM}=\frac{mV_{\infty} M}{m+M} \approx \frac{p_{LS} M}{\sqrt{s}}$, $p_{LS}=mV_{\infty}$ -- импульс налетающей частицы в лабораторной системе, $s=m^2+2EM+M^2$ -- мандельстамовская переменная, а $E$ -- полная энергия налетающей частицы в лабораторной системе.

При выводе выражения (3) полагалось $\hbar=1$, $c=1$. В общем случае необходимо умножить (3) на $(\hbar c)^2$ и учитывать заряды налетающей частицы и мишени $z$ и $Z$:

\begin{equation} \begin{aligned} \label{dSigmadOmegaCMFin} \frac{d \sigma}{d \Omega_{CM}} = \left( \frac{2 \mu \alpha z Z}{ 4 p_{CM}^2 } \right)^2 \frac{1}{\sin^4{ \frac{\chi_{CM}}{2} }} \cdot \left( \hbar c \right)^2. \end{aligned} \end{equation}

При учёте того, что $-t=2 p_{CM}^2 \left( 1 - \cos \chi_{CM} \right)=4 p_{CM}^2 \sin^2 \frac{\chi_{CM}}{2}$, выражение (3) в системе центра масс может быть записано в виде

\begin{equation} \begin{aligned} \label{dSigmadOmegaCMt} \frac{d \sigma}{d \Omega_{CM}} = \left( \frac{2 \mu \alpha z Z}{ t } \right)^2 \cdot \left( \hbar c \right)^2, \end{aligned} \end{equation}

который и будет использоваться в дальнейшем.
